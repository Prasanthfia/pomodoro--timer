# 🍅 Pomodoro Timer

A focused productivity timer based on the Pomodoro Technique, with a bold dark theme and animated ring countdown.

## Live Demo
Open `index.html` in any browser — no installation needed.

## Features
- Three modes: Pomodoro (25 min), Short Break (5 min), Long Break (15 min)
- Animated SVG ring countdown
- Play / Pause / Reset / Skip controls
- Session dots tracking up to 4 pomodoros
- Completed sessions and total minutes counter
- Browser tab title updates with live countdown

## Built With
- HTML5
- CSS3 (SVG Animation)
- Vanilla JavaScript

## The Pomodoro Technique
1. Work focused for 25 minutes
2. Take a 5-minute short break
3. After 4 pomodoros, take a 15-minute long break
4. Repeat!
